章节一：末日之后

  • 小说

朱柯，一名普通的高等年级学生，在公元2099年的某天清晨，他如同往日一般，有些散漫的踏上了去学校的路上。

路上风景不错，但天还蒙蒙，红花绿叶在雾霾中无法显露本有颜色，空气还是一如既往的污浊。

朱柯戴了口罩。

和其他人一样，朱柯挤在小小的高速地铁里面，这些设施已然免费开放给居民使用——结果就是每天都被挤得满满当当，至少在朱柯所在的城市是如此。

一首歌的时间，朱柯便下了地铁，踱步走入直梯到达地表，随后。

不寻常的事情发生了。

剧烈的闪光先是利剑一般的穿刺而来，给恰好望向那边的人的眼睛给予致命的伤害——足矣令人致盲的光芒，在远处持续了两到三秒。

朱柯很幸运，他背对闪光，但光线于地面，墙壁，远处楼层的漫反射，也足够让他晕眩一会。

“干…这啥啊…”

朱柯被晃到了眼睛，下意识的用手撑住了墙壁，不过一会发生的事表明，这似乎不是一个好主意。

比人群的尖叫更猛烈的巨大气浪飞快推进而来，黄色的尘土席卷着地表的一切——车辆，私人飞机，以及所有的建筑。尽管那些依托电子设备的车辆飞机已经因为电磁辐射而通通失灵，不过这次的风暴则是将这一切彻底的压垮撕碎。

朱柯理所当然的被吹飞了，如同那些碎石瓦砾一起，在空中翻了几圈，最后摔到了水池里面，当场失去了意识。

不过这表明朱柯是幸运的，水不是很深，不足以淹没他，反而起到了恰到好处的缓冲。也正因如此，朱柯并没有受到什么严重的伤害，顶多也就是昏迷一会。

他的黑色短发被水浸湿，同全身上下一样湿漉漉，因为戴了口罩，朱柯并未因为吸入粉尘而窒息。

同样的爆炸再次出现了，如同刚刚一般，强光和冲击波远远的扩散着，一波又一波。

——是核战争。

因为媒体的遮掩，人类并没有察觉到国际关系已经如此严峻。但毫无疑问，能够引发核弹引爆的，绝对不是什么小的战争。

事实上，战争在1小时内已经结束了，几乎完全毁灭了人类历史的成就。

瓦砾，碎屑，钢筋，车辆的碎片残渣，燃烧的汽油，以及被建筑物残骸尖角所洞穿的男人，造成这一切的，正是人类对核武器储备的全部当量。

朱柯所在的城市只遭受了一枚核弹的破坏，但当量相当之大，聚变产生的高温与气压几乎把一切碾碎，然后散乱的沉积地面，扬起的灰尘飞向了天空，把原本的蔚蓝染成了死气的灰白。而在爆炸中心的建筑，汽车，人类，则是被瞬间气化，全无生存可能。

朱柯距离爆炸，仅10km。

爆炸后的辐射粉尘大概会在5-6分钟后落下。

朱柯醒来了，身体冷冰冰的，因为一直浸没冷水里面，他有些费力的爬了起来。

“靠…这是什么情况….”

朱柯还是有些不了解现在的情况，脑袋有些麻木——是撞击造成的轻微脑震荡。

随后他便看到了末日的景色。

放眼望去，遍地尽是残骸，其中混杂血浆与泥土的暗红混合物，以及人类的断臂，残指一类的血腥。

朱柯明白了，现在的情况非同寻常，他是万千人中最幸运的，只受了擦伤，而刚刚的爆炸，毫无疑问，那是氢弹。

“绝了…”

不过朱柯挺冷静的，他在世上没有挂念的人或物，他的父母早就因病去世了，所以朱柯现在的目标就只有一个。

——活下去。
